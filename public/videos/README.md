# 视频文件说明

请将你的项目视频文件放在此目录下，并使用以下命名：

## 高德项目视频（2022-2025）
- `gaode-hd-map.mp4` - 高德高精地图-感知算法演示视频
- `gaode-vlm-indoor.mp4` - 基于VLM的室内智能引导演示视频
- `gaode-ugc-wiki.mp4` - 高德UGC内容-维基百科演示视频

## 通义实验室项目视频（2025-2026）

## 视频格式建议
- 格式：MP4（H.264编码）
- 分辨率：1920x1080 或 1280x720
- 时长：建议30秒-2分钟
- 文件大小：建议小于50MB

## 如何添加视频
1. 将视频文件复制到此目录 `/public/videos/`
2. 确保文件名与上方列出的完全一致
3. 重启开发服务器或重新构建项目

## 注意事项
- 视频文件不会包含在Git仓库中（已添加到.gitignore）
- 部署时需要确保视频文件也被上传到服务器
- 如果视频较大，可以考虑使用视频托管服务（如阿里云OSS、腾讯云COS等）
